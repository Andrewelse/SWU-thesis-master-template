# 西南大学研究生学位论文 LaTeX 模板

本模板基于 [GitHub - zoam/xmu-thesis-grd: 厦门大学研究生学位论文 LaTeX 模板](https://github.com/zoam/xmu-thesis-grd)) 按照'西南大学专业学位硕士论文规范与评价标准-202407.doc'、'D5.硕士学位论文封面格式.doc'进行了修改。本模板需要安装 `TeX Live`，并用 `XeLaTeX` 进行编译。
